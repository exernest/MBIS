# MBIS是RaE的MAA基建排班表  
目前仅有153排班表  
153排班表需勾选 MAA-基建设置-不将已进驻的干员放入宿舍  
本表为12+7+5排班  
建议凌晨5点 下午17点 午夜0点换班
